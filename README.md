## 📊 Dashboard Overview

This Excel-based **Personal Expense Dashboard** provides a comprehensive view of spending habits using 1,000 transaction records. With a sleek **dark neon UI**, it delivers instant financial insights through dynamic charts and interactive filters.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📈 **Monthly Trend Line** | Tracks expense fluctuations across all 12 months |
| 🍩 **Category Donut Chart** | Visualizes spending split across 6 categories |
| 📊 **Payment Mode Bar Chart** | Compares UPI, Cash, and Card spending at a glance |
| 🔢 **KPI Summary Cards** | Total Expense, Avg Expense, Total Transactions, Max Expense |
| 🔽 **Interactive Slicers** | Filter by Month, Payment Mode, and Category dynamically |
| 💡 **Key Insight Panel** | Auto-highlights the largest expense category |

## 🚀 Getting Started

### Prerequisites
- Microsoft Excel 2016 or later (for full slicer + chart support)
- Macros enabled *(if applicable)*

### Usage

1. **Clone or Download** the repository
   ```bash
   git clone https://github.com/yourusername/personal-expense-dashboard.git
   ```

2. **Open** `Personal_Expense_Dashboard.xlsx` in Microsoft Excel

3. **Interact** using the slicers on the right panel:
   - Filter by **Month** (Jan – Dec)
   - Filter by **Payment Mode** (Card / Cash / UPI)
   - Filter by **Category** (Bills / Entertainment / Food / Rent / Shopping / Travel)

4. All **charts and KPI cards update automatically** based on your filter selections.
